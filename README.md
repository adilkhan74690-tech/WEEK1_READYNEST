# FormNexus

FormNexus is a form-building platform that allows users to create, publish, and manage custom forms through a simple and modern interface.

The project was developed as part of my internship journey to gain hands-on experience with full-stack web development, Firebase integration, authentication systems, and real-world deployment workflows.

## Features

- Secure user authentication
- Dynamic form creation
- Multiple field types
- Form publishing and sharing
- Response collection
- Analytics and insights
- Responsive user interface
- Real-time data storage using Firestore

## Built With

- Next.js
- React
- TypeScript
- Firebase Authentication
- Cloud Firestore
- Tailwind CSS
- ShadCN UI
- Vercel

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Environment Variables

Create a `.env.local` file and configure your Firebase credentials before running the project.

## About This Project

This project was built to explore modern web development practices and gain practical experience in building scalable applications with Firebase and Next.js.

Throughout development, the focus was on creating a clean user experience while learning authentication flows, database management, deployment pipelines, and application architecture.
