# **App Name**: FormNexus

## Core Features:

- Intelligent Layout Architect: A sophisticated drag-and-drop UI workspace for creating dynamic form schemas with built-in validation rules using Shadcn UI.
- AI Draft Assistant: An AI-powered tool that interprets text prompts to automatically generate logical form structures, including field types and labels.
- Firestore Data Backbone: A secure Firestore integration to manage real-time form publishing, user configurations, and multi-tenant schema storage.
- Public Capture Engine: High-performance server-rendered public pages using slugs for zero-latency response collection and secure data submission.
- Live Insights Dashboard: Real-time visualization hub using Recharts to track submission trends, conversion rates, and total engagement metrics.
- Distribution Tooling Suite: Utility feature providing one-click QR code generation for every form and CSV exporting for external data analysis.
- Profile & Identity Portal: Robust authentication and user management system powered by Firebase, allowing personalized form ownership and storage.

## Style Guidelines:

- Primary Color: Professional Teal (#0D9488), chosen for its precise and modern SaaS appeal.
- Background Color: Ultra-light minimalist teal-tinted white (#F7FCFC) to provide a crisp, clean professional workspace.
- Accent Color: Deep Emerald (#2EB872), used sparingly for primary success actions and conversion-focused buttons.
- Primary Font: 'Inter' (sans-serif) for body and UI elements, ensuring maximum objective clarity and legibility.
- Headline Font: 'Space Grotesk' (sans-serif) for high-impact branding and dashboard headings to create a technical, professional edge.
- Consistent 2px stroke weight icons that favor geometric clarity and sharp corners, mirroring the professional sidebar layout.
- A modern sidebar-driven dashboard with persistent navigation and balanced white space to minimize cognitive load.
